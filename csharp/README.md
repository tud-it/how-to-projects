# How to create it

## Extentions
- [C#](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp)

## Dependencies
```bash
sudo pacman -Sy dotnet-sdk
```

## Settings
- [.vscode/settings.json](.vscode/settings.json)
- [.editorconfig](.editorconfig)

## Run
##### Create
```bash
cd ./hello_world
dotnet new console
```
##### Run
```bash
dotnet run
```
